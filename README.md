# 쿠팡파트너스 황금키워드 분석기

구글 블로거를 위한 쿠팡 파트너스 키워드 분석 도구입니다.

## 기능

- 🔍 100개 황금키워드 실시간 분석
- 📊 카테고리별 필터링 (가전/디지털, 생활용품, 식품 등)
- 🎯 메인 키워드 입력 시 서브키워드 15개 자동 발굴
- ✍️ 블로그 포스팅 플랜 자동 생성
- 📥 엑셀 다운로드 (구글 스프레드시트 호환)

## Vercel 배포 방법

### 방법 1: Vercel CLI 사용

```bash
# Vercel CLI 설치
npm install -g vercel

# 프로젝트 디렉토리에서 배포
vercel

# 프로덕션 배포
vercel --prod
```

### 방법 2: GitHub 연동

1. GitHub에 코드 푸시
2. Vercel.com 접속
3. "Import Project" 선택
4. GitHub 저장소 연결
5. 자동 배포 완료!
